Thank you for downloading the Quality Check files. This file will provide important information to prevent Quality check rejects for products in your consignment.
Please carry out the steps below:-

1.Download file named ‘Labels’. If your consignment has products from multiple verticals, this file will provide all fields for each vertical that should be present
on your product label. In case any field is missing, your products will get rejected during the quality check process
2.Open the individual files which have names starting with ‘Quality_Check_’. For each vertical,  this file will give you all the fields that will be used during
Quality Check. You need to check the fields and the values existing in Flipkart catalog for each listing. In case of any discrepancies between actual product and information
in our systems, you can make changes in the catalog by using ‘Edit Catalog’ option. Please mail 'ss@flipkart.com' if you face any issues in using Edit Catalog option.
