MMS 3 Tools - V25 Final Updated Notes

IMPORTANT FOR TOOL 2:
PDF preview/export is NOT the final print proof.
Final barcode label printing must be checked with PRN/direct print.

Why:
- PDF uses computer/PDF fonts and Windows scaling.
- PRN uses native printer TSPL/TSC commands like BarTender.
- Barcode scanning is most reliable from PRN/direct print.

Recommended workflow:
1. Open Tool 2.
2. Upload Excel/CSV.
3. Select PRN Print Mode = bartender_readable.
4. Select Printer.
5. Generate PRN & Print only 1 row / 2 labels first.
6. Check barcode scan and text visibility.
7. If OK, print full batch.

Category support:
The package includes category PRN/template mapping created from the uploaded BarTender PRN references and QC/CSV files.
Categories covered include: bracelet/bangle, pendant/locket, key chain, earring, jewellery set, necklace chain, car hanging ornament, ring, showpiece/figurine, and brooch.

Do not print 100 labels without one PRN test print.
